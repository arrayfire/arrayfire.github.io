Detailed Description

approx2 performs interpolation on data along the first and second dimensions.

It has three options for the type of interpolation to perform:

  • Nearest neighbor - AF_INTERP_NEAREST
  • Linear interpolation - AF_INTERP_LINEAR
  • Bilinear interpolation - AF_INTERP_BILINEAR
  • Cubic interpolation - AF_INTERP_CUBIC
