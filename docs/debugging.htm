Debugging ArrayFire Issues

Using Environment Variables

  • AF_PRINT_ERRORS=1 : Makes exception's messages more helpful
  • AF_TRACE=all: Print ArrayFire message stream to console
  • AF_JIT_KERNEL_TRACE=stdout: Writes out source code generated by ArrayFire's JIT to the specified target
  • AF_OPENCL_SHOW_BUILD_INFO=1: Print OpenCL kernel build log to console

Tips in Language Bindings

C++

  • af_print_mem_info("message", -1);: Print table of memory used by ArrayFire on the active GPU

Python

  • arrayfire.device.print_mem_info("message"): Print table of memory used by ArrayFire on the active GPU
